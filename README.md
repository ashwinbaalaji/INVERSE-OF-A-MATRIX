# INVERSE-OF-A-MATRIX
## Aim:
To write a python program to find the inverse of a matrix
## Equipment’s required:
1. 	Hardware – PCs
2. 	Anaconda – Python 3.7 Installation / Moodle-Code Runner
## Algorithm:
### Step1 : first import numpy using import numpy as np
### Step 2: then create a array with the values
### Step 3: next find the inverse of the matrix using np.linalg.inv()
### Step 4: we found the inverse of a array using numpy

## Program:
```
#Program to find the inverse of a matrix.
#Developed by: ASHWIN BAALAJI V K
#RegisterNumber:25011987
import numpy as np
a=np.array([[1,0,3],[-1,2,-2],[2,3,-1]])
rank=np.linalg.inv(a)
print(rank)
```
